    • 1. 文档概述

    1.1 项目背景

    泰小虎是一个面向 45–70 岁中老年用户 的 AI 健康导购助手，基于扣子（coze.cn）平台构建。管理后台服务于运营人员、产品经理和技术管理员，支持 AI 智能体的配置管理、知识库维护、用户运营和数据分析。

    1.2 目标用户角色

    | 角色 | 职责 | 权限范围 |

    |------|------|----------|

    | 超级管理员 | 全局配置、权限管理、系统设置 | 全部功能 |

    | 运营管理员 | 知识库管理、用户管理、对话监控、数据看板 | 除系统设置外全部功能 |

    | 数据分析师 | 数据看板、Case 分析、导出 | 数据看板、Case 管理、导出 |

    | 内容编辑 | 知识库编辑、Prompt 编辑 | 知识库管理、Agent 配置（只读） |

    2.3 设计风格

    • 主色调: 温暖橙色系（#FF6B35），呼应"泰小虎"品牌形象，传递亲切、可信赖感
    • 辅助色: 深蓝（#1A365D）用于导航和标题，浅灰（#F7FAFC）用于背景
    • 字体: 系统默认字体栈，正文 14px，标题 16–20px
    • 圆角: 卡片 8px，按钮 6px
    • 间距: 统一使用 8px 倍数（8/16/24/32px）

    2.4 响应式断点

    • 桌面端: >= 1280px（完整三栏布局）
    • 平板端: 768px–1279px（侧边栏可折叠）
    • 移动端: < 768px（暂不支持，管理后台以桌面端为主）

    3.3 交互说明

    | 交互场景 | 行为描述 |

    |----------|----------|

    | 账号密码登录 | 用户输入账号和密码，点击"登录"按钮或按 Enter 键提交。前端校验非空后，调用后端登录接口，成功后跳转至仪表盘页（P02），失败则表单下方显示红色错误提示 |

    | 记住登录状态 | 勾选后，使用 Refresh Token 机制保持 7 天免登录。Token 存储于 HttpOnly Cookie |

    | 企业 SSO 登录 | 点击后跳转至企业统一认证页面，认证成功后回调并自动登录 |

    | 忘记密码 | 点击后弹出模态框，输入注册邮箱，发送重置链接。链接 30 分钟内有效 |

    | 登录安全 | 连续 5 次失败后锁定账号 30 分钟，并显示剩余锁定时间。支持图形验证码（第 3 次失败后出现） |

    | 会话超时 | 登录后默认 2 小时无操作自动退出，退出前 5 分钟弹出提示框，用户可选择"继续使用"或"退出登录" |

    14.3 交互说明

    | 交互场景 | 行为描述 |

    |----------|----------|

    | 告警规则编辑 | 点击"编辑"按钮，弹出告警规则编辑模态框：① 选择告警指标（满意度/Bad Case 率/响应时长/同步状态/对话量）② 设置触发条件（阈值 + 持续时间 + 环比条件，支持组合）③ 选择通知方式（站内消息/邮件/企业微信/短信，多选）④ 选择通知人员（按角色或指定个人）⑤ 设置静默期（避免重复告警） |

    | 告警测试 | 编辑告警规则时，可点击"发送测试告警"按钮，向配置的通知渠道发送一条测试告警，验证通知链路畅通 |

    | 灰度策略配置 | 调整灰度参数后实时预览影响范围（显示预计覆盖用户数）。保存后立即生效，影响后续所有灰度发布 |

    | 权限编辑 | 点击"编辑权限"，弹出权限矩阵表格（角色 x 功能模块），通过勾选框配置每个角色的功能访问权限。支持设置"只读"权限 |

    | 成员管理 | 添加成员时输入姓名、账号、选择角色和部门。支持批量导入成员（Excel 模板下载）。移除成员需二次确认 |

    | 连接测试 | 点击"测试连接"按钮，验证 Coze API 和火山引擎 RAG 的连接状态，显示响应时间和连接状态 |

    | 配置保存 | 系统参数修改后需点击"保存所有配置"，弹出确认面板展示变更摘要。关键配置变更（如 API Key）需超级管理员二次密码验证 |

    17.4 交互说明

    | 交互场景 | 行为描述 |

    | 新建配置 | 点击"新建配置"按钮，弹出配置编辑模态框，填写配置名称、类型、联系方式、服务时间、触发场景、显示文案 |

    | 编辑配置 | 点击配置行的"编辑"按钮，弹出配置编辑模态框，预填充当前配置信息 |

    | 类型切换 | 选择不同的配置类型（电话/二维码/链接），联系方式输入区域动态切换为对应的输入组件 |

    | 图片上传 | 二维码类型时，支持点击选择图片或拖拽上传，支持预览、删除、重新上传 |

    | 触发场景选择 | 多选触发场景，支持选择预设场景或添加自定义场景。选择"所有转人工场景"时，其他场景选项自动选中 |

    | 优先级配置 | 当多个配置匹配同一场景时，按优先级选择。优先级数字越小，优先级越高 |

    | 启用/禁用 | 点击"启用"或"禁用"按钮，切换配置状态。禁用后该配置不会在对话中展示 |

    | 查看统计 | 点击"查看统计"，弹出统计面板，显示该配置的转接次数、成功率、响应时长等数据 |

    | 配置验证 | 保存前验证配置完整性：电话号码格式、链接格式、图片大小等 |

    18.1 通用交互模式

    | 模式 | 说明 |

    |------|------|

    | 加载状态 | 数据加载时显示骨架屏（Skeleton），加载时间超过 3 秒显示进度提示。局部刷新使用 Loading 遮罩 |

    | 空状态 | 列表无数据时显示空状态插图 + 引导文案 + 操作按钮（如"创建第一条数据"） |

    | 错误状态 | 接口错误显示错误提示条，包含错误原因和"重试"按钮。表单校验错误在对应字段下方显示红色提示 |

    | 确认操作 | 删除、下线、回滚等高风险操作需二次确认弹窗，弹窗明确说明操作影响 |

    | 操作反馈 | 成功操作显示绿色 Toast 提示（3 秒自动消失），失败操作显示红色 Toast 并提供"查看详情"链接 |

    | 表单保存 | 表单页面支持 Ctrl+S 快捷键保存。未保存离开页面时弹出"确认离开？"提示 |

    | 数据导出 | 导出操作异步执行，导出按钮变为进度状态。导出完成后自动下载并显示成功提示 |

    18.4 数据安全规范

    | 规范 | 说明 |

    |------|------|

    | 数据脱敏 | 用户手机号、身份证号等敏感信息默认脱敏显示（中间 4 位用 * 替代） |

    | 操作审计 | 所有数据变更操作记录审计日志（操作人、时间、操作内容、IP 地址） |

    | 权限控制 | 前端路由级 + 后端接口级双重权限校验 |

    | 会话安全 | Token 存储于 HttpOnly Cookie，支持强制下线（管理员可踢出指定用户） |

    | 数据加密 | API Key 等敏感配置数据加密存储，前端展示时脱敏 |
