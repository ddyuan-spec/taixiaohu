📝 提示词配置功能

需求文档 | 优先级 P0 | 版本 v1.0

📌 需求概述

提供后台管理界面，支持运营人员动态配置和修改各模块的提示词（Prompt），无需修改代码即可调整智能体的回复风格、规则和行为。

🎯 功能目标

  • 支持查看和编辑所有16个模块的提示词
  • 支持提示词版本管理（保存历史版本，可回滚）
  • 支持实时预览修改后的效果
  • 支持一键恢复默认提示词

📋 功能清单

功能点说明优先级
提示词列表展示所有16个模块的提示词，按分类展示P0
提示词编辑支持在线编辑提示词内容，带语法高亮P0
保存/发布保存修改后的提示词，可选择立即生效或草稿P0
版本历史记录每次修改的历史版本，支持对比和回滚P1
实时预览修改后可快速测试效果（输入测试对话）P1
恢复默认一键恢复提示词到默认版本P1
导入/导出支持批量导入导出提示词（JSON格式）P2

🏗️ 页面设计

左侧导航栏

  • 全局人设（System Prompt）
  • 总控决策（Orchestrator）
  • 安全守卫（Safety Guard）
  • 意图识别（Intent Router）
  • 画像模块（抽取/收集/追问/关联人切换）
  • 业务模块（症状收集/推荐引擎/产品咨询/知识问答）
  • 服务保障（边界处理/客服转接/免责声明/欢迎语）

右侧编辑区

  • 顶部：模块名称 + 当前版本号 + 保存/发布按钮
  • 中部：文本编辑区（支持Markdown语法高亮）
  • 底部：功能说明（该提示词的用途和规则）

⚠️ 注意事项

  • 修改提示词后需要点击"发布"才会生效
  • 保存为草稿时不会影响线上对话
  • 恢复默认操作不可撤销，建议先保存当前版本
  • 提示词内容需符合安全规范，不得包含违规内容
